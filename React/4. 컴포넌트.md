## 1. 클래스형 컴포넌트
   - 컴포넌트를 선언하는 방식은 두 가지 -> 함수형 컴포넌트, 클래스형 컴포넌트
   - 클래스형 컴포넌트
   ```javascript
    import React, {Fragment} from 'react';
    import { Component } from 'react/cjs/react.development';

    class App extends Component {
      render(){
        const name = 'react';
        return <div className = "react">{name}</div>
      }
    }
    export default App;
   ```
